# MapApp

This project was built using Angular, leaflet, usgeojson api, and bootstrap

## Getting Started

Run `npm install` to install the dependencies.

```
npm install
```

Run `ng serve` to start the server.

```
ng serve
```

Navigate to `localhost:4200` to view the app
